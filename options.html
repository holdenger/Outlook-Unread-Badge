Outlook Unread Badge

Counts unread messages from Favorites for app icon badge and title updates.

Disclaimer

Only folders listed in Favorites are included in badge calculation. Folders outside Favorites are ignored, and custom include/exclude rules apply only to folders that are in Favorites.

Title Counter

Show unread count in Outlook page title (for example: (2) Mail - ...).

App Icon Badge

Show unread count on Outlook app icon badge. By default, this is disabled on Windows.

  • Show unread count on app icon

Built-in Exclusions

Disable any default exclusion rule below. When enabled, that folder type is excluded from badge counting. These built-in rules are language-agnostic.

    Custom Folder Rules

    Add exact folder name and choose whether to include or exclude it. Rules are evaluated only for folders present in Favorites; custom include rules override built-in exclusions only for that exact folder name (case-insensitive, diacritics-sensitive).
